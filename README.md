My personnal and everyday vocal assistant.
====================================================
It can :

- Parse mail
- Detect proximity via bluetooth
- Prompt for search terms.
- Delay prompts and announcements until users are nearby.
- Announce and send notifications if something was found while parsing mail.
- Has a GUI with options 


Requirements:
===================================================================================
Everything in requirements.txt

xcblib for linux : sudo apt install libxcb-cursor0

google console oauth secrets at the root in /secrets/google_secrets.json 

the prompt sound must be generated by running /root/controllers/Sound/wav_to_bytes.py
